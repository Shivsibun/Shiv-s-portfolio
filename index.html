Hotel Booking

The Hotel Booking Web App is a user-friendly platform that allows customers to search, compare, and book hotels online. It provides features like hotel listings with images, room availability, pricing, and amenities. Users can filter hotels by location, budget, ratings, and facilities to find the best match. The app typically includes secure user authentication, booking confirmation, and payment integration for a smooth reservation process. On the admin side, hotel managers can manage rooms, update availability, set prices, and view booking details.

Slack Connect

A full-stack Slack Connect app that enables users to log in via Slack OAuth and schedule messages to Slack channels. It includes a React frontend and an Express backend with TypeScript. The app uses ngrok for local tunneling during development and is deployed on Render (backend) and Vercel (frontend). It handles secure OAuth flows.

A simple web app that allows users to create and customize CSS gradients. Users can select colors, adjust angles, and see a live preview of their gradient. The app generates the corresponding CSS code for easy integration into projects.

Food Circle

Food Circle is a web application that enables users to explore restaurants, browse menus, and order food online in a simple and efficient way. The platform provides features like restaurant and dish search, filters for cuisine and pricing, user authentication, and a smooth checkout process with order tracking. On the admin side, restaurant owners can manage menus, update pricing, and monitor incoming orders.
